# SARVAX Agent Capabilities

## HR Agent
**Function:** Automates the full top-of-funnel hiring process
**Processes Automated:**
- Job description creation
- Multi-channel candidate sourcing
- Application intake and qualification screening
- Shortlist generation with scoring
- Interview scheduling and calendar coordination
- Candidate communication and status updates

**KPIs It Impacts:**
- Time-to-shortlist
- Time-to-hire
- Cost-per-hire
- HR team bandwidth utilisation

**Best for:** Staffing agencies, growing companies with 10+ active roles, HR teams under hiring pressure

---

## Sales Agent
**Function:** Automates top-of-funnel B2B sales outreach
**Processes Automated:**
- ICP list building and enrichment
- Personalised outreach sequencing (email + LinkedIn)
- Follow-up cadence management
- Meeting booking and CRM logging
- Lead qualification scoring

**KPIs It Impacts:**
- Outreach volume per SDR
- Response rate
- Meetings booked per week
- Pipeline velocity

**Best for:** B2B SaaS, consulting firms, agencies building outbound pipeline

---

## Ops Agent
**Function:** Automates internal coordination and operational workflows
**Processes Automated:**
- Status reporting and stakeholder updates
- Task assignment and follow-up
- Document generation (SOPs, briefs, summaries)
- Data entry and cross-system syncing
- Meeting prep and post-meeting action tracking

**KPIs It Impacts:**
- Time spent on coordination tasks
- Response latency on internal requests
- Report generation time
- Operational cost per function

**Best for:** Operations teams, COO offices, project management functions
